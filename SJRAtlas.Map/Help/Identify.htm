Identify features on the map

If you want to know about a feature on the map, you can get more information about it.

To identify a feature

  1. In the Toolbar, click the Identify button. . The button changes to a pressed appearance . (If no Identify button is visible, then the Identify function is not available in this application.)
  2. Click on the feature you want to identify. An identify icon will be placed on the map at the location clicked. After a few moments, an entry for the Location should appear in the Results section of the Console. The Location will display the coordinates of the location clicked.
  3. Expand the location by clicking on the plus image to the left of the coordinates. This displays names of layers where features were found.
  4. Expand a layer to display features found in that layer by clicking on the plus image next to the layer. Then expand feature details as desired.
  5. See Using Results for more on using the Results window.
